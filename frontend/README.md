# Frontend

Frontend source lives in `src`. TanStack Start is configured with `srcDirectory: "frontend/src"` in `../vite.config.ts`.

Intended production organization:

- `components`
- `pages`
- `layouts`
- `services`
- `hooks`
- `utils`
- `assets`

The current implementation follows those boundaries inside `src`:

- `src/components`
- `src/routes`
- `src/lib`
- `src/hooks`
- `src/integrations`
